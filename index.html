IP Address Notation

IPv4 addresses are typically written in decimal (e.g., 192.168.1.1) but are processed as binary by computers. Converting between these representations helps in understanding subnet masks and network configurations.

IPv4 (Decimal): 192.168.1.1
Binary: 11000000.10101000.00000001.00000001

File Permissions

Unix file permissions are represented in octal notation, where each digit corresponds to a set of read, write, and execute permissions for the owner, group, and others.

Permission String: -rwxr-xr--
Binary: 111101100
Octal: 754

Memory Addressing

Hexadecimal numbers are commonly used to represent memory addresses in computing, making large binary values more readable for programmers.

Memory Address (Hex): 0x7FFEE400
Decimal: 2,147,414,016

Color Codes

Web colors are expressed in hexadecimal notation (e.g., #FF5733), where each two digits represent the amount of red, green, and blue in the color.

Hex Color: #3B82F6
RGB (Decimal): 59, 130, 246
